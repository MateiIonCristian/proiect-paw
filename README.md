# 🌟 ReviewHub - Platformă de Recenzii Firme

**ReviewHub** este un proiect de tip portal web care permite utilizatorilor să caute firme, să vizualizeze rating-uri și să posteze recenzii despre experiențele lor. Proiectul pune accent pe o interfață curată, modernă și ușor de navigat.

---

## 🚀 Funcționalități Actualate (Front-end)
Aplicația cuprinde **8 pagini web distincte**, toate având un layout unitar și responsiv:

1.  **Acasă (`index.html`):** Pagina de start cu motor de căutare și categorii.
2.  **Catalog Firme (`firme.html`):** Lista tuturor companiilor înregistrate.
3.  **Detalii Firmă (`detalii-firma.html`):** Vizualizarea profilului unei firme și a recenziilor primite.
4.  **Adaugă Recenzie (`adauga-recenzie.html`):** Formular interactiv pentru postarea de feedback.
5.  **Top Clasament (`top-firme.html`):** Ierarhia firmelor în funcție de performanță.
6.  **Categorii (`categorii.html`):** Organizarea firmelor pe domenii de activitate.
7.  **Despre Noi (`despre.html`):** Informații despre misiunea platformei.
8.  **Profil Utilizator (`profil.html`):** Panoul personal unde apare numele (**Popescu Ion**) și email-ul utilizatorului conectat.

---

## 🛠️ Tehnologii Utilizate
* **HTML5 & CSS3:** Structură și stilizare personalizată.
* **Bootstrap 5:** Framework utilizat pentru layout-ul de tip Grid (12 coloane) și componente (Navbar, Cards, Buttons).
* **FontAwesome:** Pentru iconițe intuitive.
* **Mermaid.js:** Utilizat pentru proiectarea diagramei ER a bazei de date.

---

## 📊 Arhitectura Bazei de Date (ERD)
Proiectul include o planificare pentru o bază de date relațională cu **6 tabele principale**:
* `Users`, `Companies`, `Categories`, `Reviews`, `Responses`, `Reports`.
